## Overview

This project replicates a UI component based on the provided Figma design.
The goal is to match the layout, spacing, typography, colors, and interactions
as accurately as possible.

## Tech Stack

- HTML
- CSS (no framework)
- JavaScript (minimal, only for visible interactions)
- Cloudflare Pages (deployment)

## Scope & Assumptions

- The implementation strictly follows the Figma design and prototype.
- No additional behavior or business logic was added beyond what is visible in the design.

## Pixel-Perfect Notes

- All spacing, font sizes, colors, and border radii were taken directly from Figma.
- Font family: Montserrat
- Hover and checked states match the design specification.
